# 🛒 Shopping List App

אפליקציית רשימת קניות משותפת עם תכונות מתקדמות.

## ✨ תכונות

- 📝 רשימות קניות משותפות
- 👥 שיתוף עם משתמשים נוספים
- ✅ סימון מוצרים שנרכשו
- 💬 צ'אט בזמן אמת לכל רשימה
- 📸 שיתוף תמונות
- 🏷️ קטגוריות מותאמות אישית
- 🔄 שכפול רשימות
- 🔔 התראות על עדכונים
- 👤 שיוך מוצרים לחברים

## 🚀 טכנולוגיות

- **Kotlin** - שפת התכנות
- **Jetpack Compose** - UI מודרני
- **Firebase** - Backend (Firestore, Auth, App Distribution)
- **Hilt** - Dependency Injection
- **Coroutines & Flow** - Async operations
- **Material 3** - עיצוב

## 🔧 פיתוח

### דרישות
- Android Studio Hedgehog | 2023.1.1 or newer
- JDK 17
- Android SDK 34

### הרצה מקומית
1. שכפל את הפרויקט
2. פתח ב-Android Studio
3. סנכרן Gradle
4. הרץ על emulator או מכשיר
